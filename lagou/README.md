# 拉勾网数据爬取

### 数据爬取

爬取自己感兴趣的职位信息

['web前端','PHP','Android','ios','产品总监']

### 数据分析

  - 职位总数统计 （衡量职位的热门度）
  - 工作年限要求统计，生成柱状表 （衡量职位要求年限的热门度）
  - 薪水统计，与工作年限一起生成柱状表 
  - 职位要求关键字 （衡量某职位的技能要求，看哪种技术目前需求最高）
  - 福利统计 （衡量各公司的各种福利）
  - 学历要求统计 （各学历指标）
